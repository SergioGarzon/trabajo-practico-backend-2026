### Microservicio de Yahoo Finance

#### Tecnologias

* API extraida de RAPID-API
* Se utiliza la libreria de Jackson para extraer informacion de las acciones los nombres y los simbolos
* Se utiliza Prometheus para mostrar las metricas
* Se utiliza la UI de Grafana para ver metricas en forma grafica
* Se utiliza la libreria de Logger para hacer log sobre el bootstrap del proyecto
* Se agrega un banner informando sobre los integrantes del grupo
* Para realizar las pruebas de las APIs se utiliza HTTPie
* Se agrega KarateFrameworks para automatizar las pruebas de forma rapida
